# iskra-data
iskra 数据

收录、整理可靠的马克思主义著作和历史材料，统一采用机器友好的 markdown 格式。

## 目的
- 作为 Iskra 项目的 RAG 数据，以及未来以后对结构化文本格式的文献需求。
- 作为个人网站里的资料分享部分，用 Nuxt Content 管理，促进传播学习。

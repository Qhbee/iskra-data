# 给彼得格勒指挥员训练班的电报

１９１８年９月１８日彼得格勒瓦西里耶夫岛军校街３号

致军区政治委员

我向今天在红军指挥员训练班毕业并即将到军队中去担任领导工作的４００名工人同志表示祝贺。俄国和世界的社会主义革命的胜利，取决于工人以怎样的毅力管理国家，以怎样的毅力指挥为推翻资本压迫而战的被剥削劳动者的军队。因此我相信，成千上万的工人将以这４００人为榜样；有这样的管理人员和指挥人员，共产主义的胜利就会有保障。

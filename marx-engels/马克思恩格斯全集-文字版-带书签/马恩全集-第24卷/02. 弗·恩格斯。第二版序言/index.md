# ［第二版序言］

> [^1]

这里印出的第二版，基本上是按第一版原样翻印的。印刷错误改正了，若干文体上不讲究的地方纠正了，若干短的、内容重复的段落删掉了。

第三卷虽然遇到完全料想不到的困难，但手稿现在也已大体整理就绪。只要我还健康，今年秋天就可以开始付印。

# 《资产阶级和反革命》 一 文的草稿片断３３７

现在我们放下纲领，进而来谈谈纲领的执行问题，我们没有忘记，汉泽曼内阁体现的时代是：上了台的德国庸人努力扮演英国或法国资产者的角色。

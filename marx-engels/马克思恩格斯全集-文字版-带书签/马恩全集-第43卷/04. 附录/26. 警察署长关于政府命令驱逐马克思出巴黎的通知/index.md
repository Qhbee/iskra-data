# 警察署长关于政府命令驱逐马克思出巴黎３７０的通知

> 圣热尔门郊区地段一千八百四十九年
>
> 七月十九日巴黎市圣热尔门郊区地段警察署长办公室

# 给马克思先生的关于迁往摩尔比安省居住的通知

本人，圣热尔门郊区地段警察署长，

谨通知**马克思**（卡尔）先生（现年二十九岁，特利尔（普鲁士） 人，著作家，现住百合花路５５号），根据内政部长[^1]的决定，马克思应前往摩尔比安省居住，部长对该人所采取的措施本人理应遵奉。

#### 杜尔朗

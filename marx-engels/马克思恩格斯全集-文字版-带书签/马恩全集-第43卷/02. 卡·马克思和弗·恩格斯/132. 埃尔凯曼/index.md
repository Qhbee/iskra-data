# 埃尔凯曼

**科伦**４月１８日。昨天柏林第二议院对曼托伊费尔先生提出的压制结社权的法案是否应该整个加以否决的问题进行了表决。２８５ 否决法案的提案以一百三十七票对一百四十一票被否决。因此，左派仅以四票之差仍居少数。这四票属于中间派左翼，他们是**沃林根的埃尔凯曼**牧师先生和**科伦及米尔海姆农业地区**的议员。我们试问复选人和初选人，他们选举这位当时具有极端自由思想的牧师先生，难道是为了要他帮助取消他们所剩无几的公民权吗？

> 弗·恩格斯写于１８４９年４月１８日原文是德文载于１８４９年４月１９日《新莱茵报》 第２７６号

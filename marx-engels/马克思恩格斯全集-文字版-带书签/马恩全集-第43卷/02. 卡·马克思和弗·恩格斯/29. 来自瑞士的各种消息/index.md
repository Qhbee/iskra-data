# 来自瑞士的各种消息

**伯尔尼**１１月２０日。我刚从联邦军事部门的官员处获悉，**据称德意志中央政权２９已向瑞士宣战**。５５信使于昨晚到达，首府州５１立即在十一时召开了会议。现已采取措施，进行了认真的备战。又称， **有五万名帝国军队士兵集结在瑞士边境**，准备开战。

这些消息都是照传不误。尽管来源可靠，但我本人并不相信。 我想，即使帝国内阁也不致作出**这种**荒谬事情来。

> 弗·恩格斯写于１８４８年１１月２０日原文是德文载于１８４８年１１月２４日《新莱茵报》 第１５１号附刊

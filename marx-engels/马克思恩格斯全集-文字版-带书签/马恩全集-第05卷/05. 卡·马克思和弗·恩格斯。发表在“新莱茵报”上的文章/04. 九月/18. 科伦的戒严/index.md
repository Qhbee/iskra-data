# 科伦的戒严

> **２４５**

**  科伦９月２６日**。**今天我们又出版了一号没有标题的报纸**，**这是因为我们急于出报的缘故**。**我们从可靠的消息得知**：**在几点钟内**，**城市就要宣布戒严**，**市民自卫团将被解散和解除武装**，**“新莱茵报”**、**“新科伦报”２４６**、**“工人报”２４７和“莱茵守卫者”２４８将被禁止出版**， **同时将要建立军事法庭并取消在３月里争得的一切权利**。**风闻**：**市民自卫团将不容许解除它的武装**。

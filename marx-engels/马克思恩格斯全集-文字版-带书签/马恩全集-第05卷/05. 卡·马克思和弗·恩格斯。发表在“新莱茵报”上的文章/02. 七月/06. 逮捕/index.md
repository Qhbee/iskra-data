# 逮捕

**  科伦**７月３日。**办事**内阁目前的表现说明它只不过是一个警察内阁。它的第一个行动是在柏林逮捕**莫内克**先生和**弗恩巴赫**先生，第二个行动是在萨尔鲁伊逮捕炮兵下士**丰克**。现在，这些“事情”也在科伦这儿开始了。今天早晨逮捕了医师**哥特沙克**先生和退伍中尉**安内克**先生。无论是逮捕的原因或逮捕时的情况，我们都还没有得到确切的消息，因此暂时不发表意见。

工人们是十分理智的，他们不会因挑衅而贸然行动。

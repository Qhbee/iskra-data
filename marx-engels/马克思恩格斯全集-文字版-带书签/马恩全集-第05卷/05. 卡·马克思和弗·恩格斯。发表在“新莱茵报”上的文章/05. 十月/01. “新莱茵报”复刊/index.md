# “新莱茵报”复刊

**  大家为维持“新莱茵报”的存在而给予的援助（特别是在科伦）**，**使我们得以克服戒严状态所造成的*财政*困难**，**使报纸得以复刊**。**编辑委员会原有成员不变**。**不久前*斐迪南*··弗*莱里格拉特*参加了编委会**。

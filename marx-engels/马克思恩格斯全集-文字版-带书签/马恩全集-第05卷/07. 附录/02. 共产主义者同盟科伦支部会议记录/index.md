# 共产主义者同盟科伦支部会议记录２８３

*１８４８年５月１１日会议*

主席马克思问哥特沙克对同盟有什么意见和主张，现在打算对同盟采取什么态度。

哥特沙克回答，他坚持他的退出同盟的声明，因为目前局势已经发生变化，同盟的章程也必须加以修改；因为他认为现行的章程威胁着他个人的自由。同时他声明：在任何情况下，只要同盟需要，他都愿意在上述条件下给同盟以全力支持。

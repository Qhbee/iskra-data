# 辟谣

“布勒斯劳报” 第２０１号刊登了一篇柏林通讯，里面谈到骑士施纳普汉斯基２８８收买了大量的“新莱茵报”股票，所以描写他的小品文就不再出现了，因为报纸是不会攻击自己的股东的。假民主的“杜塞尔多夫日报” 认为转载这种诬蔑性的言论是自己的职责。在柏林，人们可以任意造谣，可是**西里西亚**的报纸应该知道这种言论是谎话以及为什么是谎话。这种背信弃义的诽谤可惜出现得太迟了，早在收到第２０１号“布勒斯劳报” 之前就已出版的第９２号“新莱茵报” 已经登载了上述小品文的续文。此外，“新莱茵报” 是党的机关报，它已经充分证明是收买不了的。

# 摘自《德意志—布鲁塞尔报》 关于在布鲁塞尔举行１８４６年克拉科夫起义两周年纪念的报道

２月２２日晚，布鲁塞尔黑衣修女街“古宫”的大厅象过节一样，灯火辉煌，装饰着波兰和比利时的旗帜，许多人在这里隆重举行了克拉科夫革命两周年纪念会。讲演者有：列列韦尔、卡尔·马克思、弗·恩格斯、瓦劳、律师卢布林纳等人；他们热情洋溢地论述了克拉科夫起义的纯民主主义性质。本报将在下一号详细报道这次纪念会２８４，现在只提一下，在公开的会议以后举行了宴会。有将近一百人参加，比利时和德国的民主主义者频频举杯祝贺。即席歌唱者也不乏其人。

> 载于１８４８年２月２４日《德意志—原文是德文布鲁塞尔报》第１６号

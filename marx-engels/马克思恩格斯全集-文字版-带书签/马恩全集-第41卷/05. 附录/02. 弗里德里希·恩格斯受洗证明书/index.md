# ［弗里德里希·恩格斯受洗证明书

*摘自爱北斐特福音改革派教区受洗登记簿］*

１８２１年１月１８日受洗。下巴门。 第２４号——** 弗里德里希**，１１月２８日晚９时［生］，系居住在布鲁赫的商人**弗里德里希·恩格斯**先生和妻子**伊丽莎白·毛里齐娅·** **弗兰契斯卡**（父姓**范·哈尔**）的合法婚生子。

受洗时的见证人有：老**卡斯帕尔·恩格斯**先生和**弗兰契斯卡** **·克里斯蒂娜·范·哈尔**（父姓**斯内特拉格**）夫人。

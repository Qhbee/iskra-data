# 和谐

### 致燕妮

你可见过这样美妙而神奇的景象：

两颗心灵热血交融，一齐震荡，

它们息息相通，相互依傍，

它们宛如琴瑟发出柔和的音响？

两颗心灵忽而在紫玫瑰中闪光，

忽而在柔嫩的青苔下羞怯地躲藏。

即使你跋山涉水走遍四方，

你也找不到这神奇的景象，

任何符箓都无法将它唤来，

它从未出现在洒满阳光的大地上。

这种景象的生成不靠阳光，

也从不仰赖大地的滋养。

因此，即使时光像飞鸟急速翱翔，

即使阿波罗驾驭车马飞越穹苍，

即使地老天荒，万物都岑寂消亡，

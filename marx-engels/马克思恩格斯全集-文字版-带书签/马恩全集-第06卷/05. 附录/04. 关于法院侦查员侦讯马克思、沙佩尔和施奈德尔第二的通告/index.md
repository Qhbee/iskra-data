# 关于法院侦查员侦讯马克思、 沙佩尔和施奈德尔第二的通告

**  科伦**１１月２５日。在马克思、沙佩尔和施奈德尔第二因民主主义者莱茵区域委员会的第二个呼吁书[^1]一案而受法院侦查员侦讯时，被告声明这一呼吁书是由他们起草和签署的。这一声明载入记录后，侦讯就告结束。被告没有一人被捕。这一通告可以作为对区域委员会所收到的许多信件的答复。

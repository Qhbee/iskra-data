# 德利加尔斯基对“新莱茵报”的控告

**科伦**１２月２１日。今天，**马克思**博士由于被控对“公民和共产主义者”**德利加尔斯基**先生[^1]进行诽谤，又被法院侦查员传讯。这个新的审判是对“新莱茵报”的第几次审判，这很难确定，因为这类审判太多了。

我们毕竟感到遗憾的是：德利加尔斯基先生竟对我们这样忘恩负义。要知道，他之所以在欧洲有了一点儿名声，是全仗我们谈到他的几篇文章。“公民和共产主义者”**德利加尔斯基**先生，这是多么可耻的忘恩负义呵！！这就是我们时代道德败坏的标志—— 甚至连普鲁士皇家共产主义者对别人为他效劳也毫无感激之情。

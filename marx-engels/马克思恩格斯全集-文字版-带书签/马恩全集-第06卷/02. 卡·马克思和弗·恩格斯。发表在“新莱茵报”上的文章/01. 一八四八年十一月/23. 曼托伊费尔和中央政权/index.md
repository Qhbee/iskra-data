# 曼托伊费尔和中央政权

**科伦**１１月２４日。**曼托伊费尔大臣昨天向在柏林的帝国专员们宣布**：**普鲁士政府不服从法兰克福议会关于组织一个有声望的内阁的决议６７**，**因为这涉及到国家的*对内*政策**。

这样一来，**曼托伊费尔**就同意这样一种看法：法兰克福议会关于**拒绝纳税**[^1]的决议也没有任何意义和效力，因为它只是同国家的**对内**政策有关。

不过，勃兰登堡—曼托伊费尔内阁可能帮助把**莱茵省**变成**普鲁士的对外政策问题**。

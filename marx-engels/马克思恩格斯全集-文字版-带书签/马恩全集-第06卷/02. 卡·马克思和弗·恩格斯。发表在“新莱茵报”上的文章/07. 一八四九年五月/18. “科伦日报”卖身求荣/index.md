# “科伦日报”卖身求荣

**科伦**５月１３日。我们提醒本报读者注意一下最近几天的，尤其是今天的（星期日，５月１３日）**“科伦日报”**。

的确，**“最下流的淫佚放荡”**从来没有像在我们出色的当代女人[^1]最近发表的社论和通讯中那样紧密地**同卖身求荣的无耻行径** 互相结合。

几天以前，我们还看到，**“科伦日报”的老板约瑟夫·杜蒙**先生在莱茵省市政委员会代表大会上坚毅果敢地拥护大会所通过的决议。今天我们却看到，那同一个人物通过自己的奴才勃律盖曼， 在每一句话里都无耻透顶地对起义的失败表示高兴，虽然这些起义是莱茵省市政委员会作出的那些决议的结果。

然而，正因为如此，**“科伦日报”**也就有福气成为与戒严状态同时钦定给莱茵省各城市的独一无二的报纸。

不错，同时钦定给这些城市的还有**鲜血和——*污秽***！

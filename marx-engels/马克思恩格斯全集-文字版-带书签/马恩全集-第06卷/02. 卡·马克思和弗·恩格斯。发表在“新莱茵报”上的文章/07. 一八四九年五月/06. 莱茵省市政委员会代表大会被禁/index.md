# 莱茵省市政委员会代表大会被禁

**科伦**５月２日。我们特别高兴地向读者报道，尊贵的科伦市政委员会召集的莱茵省各城市市政委员会代表大会被当局用一纸命令**禁止**了。在９月间民主派的集会被禁止３２２的时候，“善良的市民们”曾那样“悠然自得”，现在他们会感谢自己的主子和国君。民主派的集会自由在１８４８年９月被取消，至少是由于公开使用暴力的法令，即戒严令，而科伦市政委员会的集会自由却是在法制基础最时行的情况下被**一脚踢掉**的。
